# 프로젝트 개요

이 프로젝트는 git과 GitHub 실습을 위한 프로젝트입니다.

## 사용 파일

- `README.md` 파일만 사용합니다.

## 커밋 규칙

커밋 작성 시 반드시 [`docs/commit_convention.md`](docs/commit_convention.md)의 규칙을 따릅니다.

- 형식: `<type>: <subject>`
- 타입은 항상 대괄호(`[`, `]`)로 시작
- 사용 가능한 타입: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
- subject는 50자 이내, 끝에 마침표 금지, 명령문으로 작성
